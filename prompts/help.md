# Help

List all available skills and their descriptions.

## Available Skills

- `/commit` - Generate a git commit message for staged changes
- `/explain` - Explain code or concept in simple terms
- `/summarize` - Summarize text, documents, or conversations
- `/help` - Show this help message

## Usage

Type a skill command followed by optional arguments:
```
/commit fix typo
/explain path/to/file.ts
/summarize https://example.com/article
```

## Regular Conversation

For regular conversation, just type your message without a slash command.
